Notes on Terminal Commands

ls: lists all the files/folders in the current directory

cd [ name]: changes directory (ie opening a file)
cd ..: goes back one directory
cd: just typing cd by itself takes you back to the root directory

clear: erases everything in the terminal

pwd: shows the path/folders you are in, from root directory up to current directory

vi [name].txt creates a new file in the current folder. this file is edited using vi
